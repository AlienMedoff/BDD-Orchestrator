# 🚀 TON Transaction DSL

**Пиши на русском - получай TON транзакции**

> *"Я устал от сложных смарт-контрактов. Теперь любой может писать блокчейн операции простым языком."*

---

## 🎯 Что это такое?

TON Transaction DSL - это **революционный инструмент**, который превращает обычные английские фразы в **настоящие TON блокчейн транзакции**. Никакого кода - просто пишешь как человеку.

### 🌟 Магия работает так:
```bash
# Отправить TON другу
SEND 1.5 TON TO @friend

# Проверить баланс кошелька
BALANCE OF EQD...твой_кошелек

# Застейкать в DeFi
STAKE 100 TON IN StonFi FOR 30 days

# Добавить ликвидность
PROVIDE 50 TON + 500 USDT TO Dedust POOL
```

**И всё.** Никаких сложных контрактов, никакого кода, никаких знаний блокчейна.

---

## 🚀 Почему это меняет всё

### 🎯 Проблема
- **TON разработка сложная** - нужны знания смарт-контрактов
- **DeFi запутанный** - куча интерфейсов, сложный вход
- **Разработка фрагментирована** - нет единой платформы

### 💡 Наше решение
- **Простой язык → блокчейн** - пиши по-английски, получай TON транзакции
- **Универсальная среда** - одна платформа для всех TON приложений
- **Ноль кода** - любой может создавать блокчейн приложения

## 🏗️ Как это работает

```
┌─────────────────┐    ┌──────────────────┐    ┌─────────────────┐
│  Английский DSL │────│  DAG Оркестратор│────│  TON Блокчейн  │
│  "SEND TON"    │    │  (BDD Парсер)   │    │  (Исполнитель)   │
└─────────────────┘    └──────────────────┘    └─────────────────┘
         │                       │                       │
         ▼                       ▼                       ▼
┌─────────────────┐    ┌──────────────────┐    ┌─────────────────┐
│  Контекст Агента│────│  Redis Кеш      │────│  Системные вызовы│
│  (Менеджер)     │    │  (Состояние)    │    │  (Очистка)      │
└─────────────────┘    └──────────────────┘    └─────────────────┘
```

### 🧠 Магия внутри
1. **BDD Парсер** - превращает Gherkin в исполняемые шаги
2. **DAG Оркестратор** - управляет сложными потоками транзакций
3. **Менеджер контекста** - обрабатывает состояние и общение агентов
4. **Системные вызовы** - безопасный слой взаимодействия с блокчейном
5. **Сервис очистки** - механизмы отката и уборки

## 🚀 Быстрый старт

### 📋 **Что нужно**
- **Python 3.10+** с asyncio
- **Docker & Docker Compose** (опционально)
- **TON API ключ** для блокчейн операций
- **Node.js & npm** для смарт-контрактов

### 🛠️ **Установка**

#### 1. **Клонируй репозиторий**
```bash
git clone https://github.com/AlienMedoff/ton-tx-dsl.git
cd ton-tx-dsl
```

#### 2. **Установи зависимости**
```bash
# Python зависимости
pip install -r requirements.txt

# Node.js зависимости (для контрактов)
npm install
```

#### 3. **Настрой окружение**
```bash
# Копируй шаблон
cp .env.example .env

# Отредактируй
# TON_API_KEY=твой_api_ключ
# TON_MODE=MOCK|TESTNET|MAINNET
```

#### 4. **Запусти тесты**
```bash
# Все тесты
pytest tests/test_engine.py tests/test_config.py -v

# Конкретные тесты
pytest tests/test_engine.py -v
pytest tests/test_config.py -v
```

#### 5. **Запусти демо**
```bash
# Мок режим (без сетевых вызовов)
TON_MODE=MOCK TON_API_KEY=mock TON_API_ENDPOINT=https://testnet.toncenter.com/api/v2 \
AGENT_ID=main python main.py
```

## 🎮 Примеры DSL

### Базовые операции
```gherkin
Feature: Базовые TON транзакции
  Scenario: Отправить TON другу
    Given у меня есть кошелек "EQSenderAddress"
    When я отправляю "1.5 TON" на "EQFriendAddress"
    Then транзакция должна быть подтверждена
    And баланс должен быть обновлен

  Scenario: Проверить баланс кошелька
    Given у меня есть кошелек "EQMyWallet"
    When я проверяю баланс
    Then я должен увидеть текущий баланс
```

### Продвинутые DeFi операции
```gherkin
Feature: DeFi операции
  Scenario: Добавить ликвидность
    Given у меня есть кошелек "EQMyWallet"
    When я предоставляю "100 TON" и "500 USDT" в "EQPoolAddress"
    Then я должен получить LP токены
    And пул должен иметь увеличенную ликвидность

  Scenario: Застейкать токены
    Given у меня есть кошелек "EQMyWallet"
    When я стейкаю "50 TON" в "EQStakingPool" на "30 дней"
    Then я должен начать получать награды
    And мой стейк должен быть заблокирован
```

## 🎛️ Режимы TON

| Режим | Что происходит |
|-------|--------------|
| **MOCK** | Нет сетевых вызовов, состояние в памяти |
| **TESTNET** | Реальный TON тестнет, безопасное тестирование |
| **MAINNET** | Реальные деньги — проверь всё дважды |

**Меняй TON_MODE в .env или docker-compose.yml**
